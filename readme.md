
## Run Locally

Clone the project

```bash
  git clone https://github.com/prateek1998/demo-app
```

Go to the project directory

```bash
  cd demo-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
### note

Currently I am using a input file for getting the request data, I can change this To Request file
